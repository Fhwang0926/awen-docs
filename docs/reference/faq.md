# 자주 묻는 질문 (FAQ)

## 가입·로그인

### GitHub / Google 로그인이 안 돼요.

브라우저 팝업 차단을 해제했는지 확인하세요. 이미 연동된 계정이 있다면 동일 이메일로 로그인하면 됩니다.

### 이메일 인증이 필요하다고 나와요.

가입 시 입력한 이메일로 전달된 인증 링크를 클릭한 뒤 다시 로그인해 주세요.

## 서비스(호스팅)

### 배포 상태가 계속 "빌드 중"이에요.

빌드 로그를 확인해 보세요. 서비스 상세 → **관리** 탭에서 로그를 볼 수 있습니다. 의존성 설치 실패, 빌드 스크립트 오류 등이 원인일 수 있습니다.

### GitHub 저장소를 연결했는데 배포가 안 돼요.

GitHub App이 설치되어 있는지, **계정** → SNS 연동에서 GitHub이 연동되어 있는지 확인하세요. 저장소 권한도 필요합니다.

### 수동 업로드 시 지원하는 파일 형식은?

zip 파일을 업로드합니다. 웹 서비스는 빌드 결과물(예: `dist`, `build`)을 zip으로 압축해 업로드하면 됩니다.

## 결제·플랜

### 플랜을 변경하고 싶어요.

**결제 및 구독** → **플랜**에서 업그레이드/다운그레이드를 선택할 수 있습니다. 연간 결제 시 할인 혜택이 있을 수 있습니다.

### 쿠폰은 어디서 적용하나요?

**결제 및 구독** → **쿠폰**에서 보유 쿠폰을 확인하고, 결제 단계에서 적용할 수 있습니다.

## API

### API Key를 잊어버렸어요.

API Key는 생성 시 한 번만 표시됩니다. 기존 Key를 삭제한 뒤 **계정** → **API Key**에서 새로 생성하세요.

### API 호출 시 401 / 403이 나와요.

`Authorization: Bearer YOUR_API_KEY` 헤더가 올바른지, Key가 만료되지 않았는지 확인하세요. 권한(전체/읽기 전용)도 확인하세요.

## 고객지원

### 더 자세한 문의는 어디로 하면 되나요?

**고객지원** → **문의하기(QnA)**에서 질문을 등록하시면 담당자가 답변드립니다. **FAQ**와 **공지사항**도 참고해 주세요.
