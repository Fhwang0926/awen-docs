# 대시보드

대시보드는 로그인 후 처음 보이는 화면으로, **인프라 구성**과 **서비스 현황**을 한눈에 볼 수 있습니다.

---

## 화면 구성

- **인프라 구성**: 현재 구성된 웹, DB, 통합 서비스 등 인프라를 시각적으로 표시합니다. 서비스 유형별로 구성을 파악할 수 있습니다.
- **나의 서비스**: 배포된 서비스 목록과 **상태**(준비됨, 배포 중, 대기 중, 빌드 중, 실패 등)를 확인할 수 있습니다.

---

## 주요 기능

- **서비스 카드**: 각 서비스의 이름, 서비스 유형(웹/DB/WAS/통합/WAS+DB), 상태, 도메인(또는 접속 정보)이 표시됩니다.
- **새로고침**: 목록을 최신 상태로 갱신할 수 있습니다.
- **추가(서비스 만들기)**: **나의 서비스** 영역의 **서비스 만들기** 버튼(➕)을 사용해 새 서비스를 생성합니다. 생성 단계는 [빠르게 시작하기](/getting-started/quick-start)를 참고하세요.
