(나의 AI 비서는 준비 중입니다.)

---

## 서비스(호스팅)에서 사용 가능한 AI 기능

서비스 상세 페이지의 **설정** 탭 또는 **WEB 설정**·**DB** 탭 안에 **AI** 관련 설정이 있습니다. 서비스 유형에 따라 사용 가능한 항목이 다릅니다.

### 웹 서비스: 맞춤형 AI 자동 빌드 (유료)

- **위치**: 서비스 상세 → **WEB 설정** 탭 또는 **설정** 탭 → AI → 웹 호스팅 설정.
- **내용**: GitHub로 배포하는 웹 서비스에서, 저장소 소스를 AI가 분석해 **빌드 옵션**(React, Vue, Angular, Svelte 등)을 자동으로 구성하고, 그 결과로 배포합니다.
  - GitHub 저장소 + 빌드 옵션이 이미 설정된 호스팅에서 적용됩니다.
  - 활성화 시 푸시마다 AI가 소스를 분석해 최적의 빌드 구성을 적용합니다.
  - 비활성화하면 기존에 선택한 빌드 옵션만 사용됩니다.
- **플랜**: Basic 이상(유료) 플랜에서 사용할 수 있습니다.

### DB/통합 서비스: DB 패스워드 자동 갱신 (AI)

- **위치**: 서비스 상세 → **DB** 탭 또는 **설정** 탭 → AI → DB 호스팅 설정.
- **내용**: 보안 강화를 위해 DB 비밀번호를 **주기적으로 자동 갱신**합니다.
  - DB 서비스 또는 통합(Unified)·WAS+DB 서비스에서 사용할 수 있습니다.
  - 활성화 시 주기적으로 DB 비밀번호가 자동 갱신됩니다.
  - 변경 전 이메일로 사전 알림을 받을 수 있으며, 새 비밀번호는 서비스 상세(환경 변수·DB 연결 정보 등)에서 확인할 수 있습니다.
  - **다음 변경 예정** 일시를 확인하고, 필요 시 **10분 뒤 예정시간으로 설정하기** 버튼으로 다음 갱신 시점을 조정할 수 있습니다.
