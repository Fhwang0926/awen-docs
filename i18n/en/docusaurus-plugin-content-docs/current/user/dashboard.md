# Dashboard

The dashboard is the first screen after login. It shows your infrastructure and service status at a glance.

## Layout

- **Infrastructure**: Visual overview of your current setup (web, DB, unified services, etc.).
- **My Services**: List of deployed services and their status (Ready, Deploying, Failed, etc.).

## Features

- **Service cards**: Name, type, status, and domain (or connection info) for each service.
- **Refresh**: Reload the list.
- **Add (Create server)**: Use **Add** in the My Services area to create a new service.
