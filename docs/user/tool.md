# 도구

본 서비스에서는 **도구** 메뉴를 통해 IDE, SQL 등 개발·운영에 필요한 기능을 사용할 수 있습니다.

## IDE

- **도구** → **IDE**에서 웹 기반 IDE를 열 수 있습니다.
- 서비스 상세 페이지 **관리** 탭의 **IDE 열기**로 해당 서비스와 연동된 IDE에 접속할 수 있습니다.
- 코드 편집, 터미널, 파일 관리 등을 브라우저에서 수행할 수 있습니다.

## SQL

- **도구** → **SQL** 또는 서비스 상세(DB 서비스) **관리** 탭의 **SQL 열기**로 SQL 도구에 접속합니다.
- 쿼리 실행, 느린 쿼리 추적, 성능 분석, AI 분석·제안 등 DB 관리를 할 수 있습니다.

## 사용 절차 요약

1. **IDE**: 나의 서비스에서 해당 서비스를 선택 → **관리** → **IDE 열기**
2. **SQL**: DB 타입 서비스를 선택 → **관리** → **SQL 열기** (또는 도구 메뉴에서 SQL 진입 후 서비스 선택)
