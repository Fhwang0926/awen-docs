# 📖 문서에 오신 것을 환영합니다

이 플랫폼은 **AI 기반 인프라**로 웹·WAS·DB 배포와 운영을 한곳에서 할 수 있습니다.  
이 문서는 **개발자용 문서**와 **사용자 매뉴얼**로 구성되어 있습니다.

## 📑 문서 구성

| 구분 | 내용 |
|------|------|
| **📘 개발자 문서** | 빠른 시작, API 사용법, 배포 방법 |
| **👤 사용자 매뉴얼** | 대시보드, 나의 서비스(호스팅), AI 기능 가이드, 나의 AI 비서, 결제·구독, 도구, 고객지원, 계정 |
| **📌 참고** | FAQ, 변경 이력 |

왼쪽 **사이드바**에서 항목을 선택하시면 됩니다.
