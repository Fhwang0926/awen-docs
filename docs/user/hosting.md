# 🖥️ 나의 서비스 (호스팅)

**나의 서비스**에서 웹·WAS·DB 등 서비스를 **생성**하고, **배포·관리**할 수 있습니다.

## 📋 서비스 목록

- **대시보드** 또는 좌측 메뉴 **나의 서비스**에서 배포된 서비스 목록을 볼 수 있습니다.
- 각 카드에서 **상태**(준비됨, 배포 중, 빌드 중 등), **도메인**, **최근 배포** 정보를 확인할 수 있습니다.

## ➕ 서비스 생성

1. **추가** 또는 **서버 생성** 버튼을 클릭합니다.
2. **서비스 타입** 선택: 웹(정적), DB, WAS, 통합 등.
3. **소스** 설정:
   - **GitHub**: 저장소·브랜치 연동
   - **수동 업로드**: zip 파일 업로드
4. 서브도메인, 플랜(유료 시), 런타임·DB 타입 등 설정 후 **생성**합니다.

:::tip 빠른 생성
단계별 화면 흐름은 [빠르게 시작하기](/getting-started/quick-start)를 참고하세요.
:::

## 📄 서비스 상세

서비스 카드를 클릭하면 상세 페이지로 이동합니다.

| 탭 | 내용 |
|----|------|
| **개요** | 상태, 도메인, 소스, 리소스 요약 |
| **관리** | IDE 열기, SQL 열기(DB 서비스), 로그 등 |
| **설정** | 소스 변경, 환경 변수, 서비스 타입·리소스 변경 |

## 📊 상태 설명

| 상태 | 설명 |
|------|------|
| 준비됨 | 정상 배포 완료 |
| 배포 중 | 배포 진행 중 |
| 빌드 중 | 빌드 진행 중 |
| 대기 중 | 작업 대기 |
| 실패 | 배포·빌드 실패 |
| 삭제 중 | 삭제 처리 중 |
