# Dashboard

The dashboard is the first screen after login. It shows your **infrastructure** and **service status** at a glance.

---

## Layout

- **Infrastructure**: Visual overview of your current setup (web, DB, unified services, etc.).
- **My Services**: List of deployed services and their **status** (Ready, Deploying, Pending, Building, Failed, etc.).

---

## Features

- **Service cards**: Name, service type (web/DB/WAS/unified/WAS+DB), status, and domain (or connection info).
- **Refresh**: Reload the list.
- **Create service**: Use **Create service** (âž•) in the My Services area to create a new service. For the full flow, see [Quick Start](/getting-started/quick-start).
