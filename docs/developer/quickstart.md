# 빠른 시작 (요약)

5분 안에 첫 배포를 진행하는 요약 단계입니다. 단계별 화면 흐름은 [빠르게 시작하기](/getting-started/quick-start)를 참고하세요.

## 1. 로그인

웰컴 페이지에서 **시작하기** 또는 **로그인** 후, **나의 서비스**로 이동합니다.

## 2. 서비스 생성

**나의 서비스** → **추가** → **서비스형** 선택(웹/DB/WAS/통합 등) → **소스**(GitHub 또는 수동 업로드) → **도메인** → **검토 및 생성**에서 **생성**을 누릅니다.

## 3. 배포 완료 후

서비스 상세에서 **도메인**으로 접속하고, **관리** 탭에서 IDE·SQL·로그를 사용할 수 있습니다.
