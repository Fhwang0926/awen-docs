# My AI Assistant

Deploy AI agents and use chat, summarization, and task automation.

## Overview

- Create and manage AI agent services from the **My AI Assistant** menu.
- Provision agents by connecting to OpenClaw-Shopping and similar services.

## Provisioning an AI agent

1. Start creating a new agent under **My AI Assistant**.
2. Follow the steps: **Intro** → **OpenClaw-Shopping connection** → **Review** → **Install**.
3. Enter connection details (e.g. API key) and complete installation to use chat, summarization, and automation.

## Features

- **Chat**: Conversational interface with the AI
- **Summarization**: Content summarization
- **Automation**: Automated tasks based on configuration

Once an agent is deployed, it appears in **My Services** and can be managed from the service detail page.
