# 계정

**계정** 메뉴에서 프로필, 보안, API Key, 회사 정보, 알림 등을 관리할 수 있습니다.

## 프로필 (일반)

- **이름**, **이메일**, **전화번호**, **프로필 이미지**를 수정할 수 있습니다.
- **저장**으로 변경 사항을 반영합니다.
- **회원 탈퇴**는 신중히 진행해 주세요. 되돌릴 수 없습니다.

## 보안

- **비밀번호 변경**: 현재 비밀번호 확인 후 새 비밀번호로 변경할 수 있습니다.
- 비밀번호는 대소문자, 숫자, 특수문자를 포함해 설정하는 것을 권장합니다.

## API Key

- **API Key** 메뉴에서 API Key를 생성·조회·삭제할 수 있습니다.
- 자세한 사용법은 [개발자 문서 - API](/developer/api)를 참고하세요.

## 알림

- **알림** 타임라인에서 읽음/안 읽음, 우선순위별로 알림을 확인할 수 있습니다.

## 회사 정보 (기업 관리자)

- **회사 정보**에서 회사명, 코드, 사업자번호, 멤버십 레벨 등을 확인·수정할 수 있습니다.
- 사용자 관리, 회사 관리 기능은 권한에 따라 표시됩니다.

## SNS 연동

- GitHub, Google 등 외부 계정 연동을 설정할 수 있습니다.
- 연동 후 해당 서비스로 로그인하거나 저장소 연동 시 사용됩니다.
