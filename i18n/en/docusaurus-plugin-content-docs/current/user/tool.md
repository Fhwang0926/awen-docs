# Tools

Use the **Tools** menu to access IDE, SQL, and other features for development and operations.

## IDE

- Open the web-based IDE from **Tools** → **IDE**.
- From a service detail page, use **Manage** → **Open IDE** to use the IDE for that service.
- Edit code, use the terminal, and manage files in the browser.

## SQL

- Open the SQL tool from **Tools** → **SQL** or from a DB service’s **Manage** → **Open SQL**.
- Run queries, track slow queries, analyze performance, and use AI analysis and suggestions.

## Quick steps

1. **IDE**: In My Services, select a service → **Manage** → **Open IDE**
2. **SQL**: Select a DB service → **Manage** → **Open SQL** (or open SQL from Tools and then select the service)
