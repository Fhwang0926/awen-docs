# Quick start (summary)

A short path to your first deployment in a few minutes. For the full step-by-step UI flow, see [Quick Start](/getting-started/quick-start).

## 1. Log in

From the welcome page, click **Get started** or **Log in**, then go to **My Services**.

## 2. Create a service

**My Services** → **Add** → Choose **service type** (web/DB/WAS/unified etc.) → Set **source** (GitHub or manual upload) → **Domain** → **Review and create** → Click **Create**.

## 3. After deployment

Open the service detail page, visit the **domain**, and use **Manage** for IDE, SQL, and logs.
