# Account

From **Account** you can manage profile, security, API Key, company info, and notifications.

## Profile

- Edit **name**, **email**, **phone**, and **profile image**.
- Click **Save** to apply changes.
- **Account deletion** is permanent; use with care.

## Security

- **Change password**: Enter your current password, then set a new one.
- Use a mix of uppercase, lowercase, numbers, and symbols for passwords.

## API Key

- Create, view, and delete API keys under **Account** → **API Key**.
- See [Developer – API](/developer/api) for usage.

## Notifications

- View notifications in the **Notifications** timeline by read/unread and priority.

## Company (for admins)

- View and edit company name, code, business number, membership level under **Company**.
- User management and company management are shown based on permissions.

## SNS connection

- Connect external accounts (GitHub, Google, etc.).
- Use them for login and repository access.
