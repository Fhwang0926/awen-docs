# Support

From **Support** you can submit inquiries, use FAQ, read announcements, and send feedback.

## Contact (QnA)

- **Submit a question**: Submit questions and receive answers from the team.
- **QnA board**: View your questions and answers.

## FAQ

- Search and read **frequently asked questions** and answers.

## Announcements

- Check important notices and updates.
- Review announcements regularly to stay informed about service changes.

## Feedback

- Submit feedback about issues or ideas for improvement.
- Feedback is used to improve the service.
