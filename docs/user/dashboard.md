# 대시보드

대시보드는 로그인 후 처음 보이는 화면으로, 인프라 구성과 서비스 현황을 한눈에 볼 수 있습니다.

## 화면 구성

- **인프라 구성**: 웹, DB, 통합 서비스 등 현재 구성된 인프라를 시각적으로 표시합니다.
- **나의 서비스**: 배포된 서비스 목록과 상태(준비됨, 배포 중, 실패 등)를 확인할 수 있습니다.

## 주요 기능

- **서비스 카드**: 각 서비스의 이름, 타입, 상태, 도메인(또는 접속 정보)을 표시합니다.
- **새로고침**: 목록을 최신 상태로 갱신합니다.
- **추가(서버 생성)**: 새 서비스를 생성하려면 **나의 서비스** 영역의 **추가** 버튼을 사용합니다.
